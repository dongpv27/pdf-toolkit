## NOTES ON THE TRANSLATIONS

- Translations are **transcreated, not literal** — keyword phrases match how
  locals actually search (e.g. VN "chuyển ảnh sang PDF", ID "gambar ke PDF",
  HI "इमेज से PDF").
- "PDF" is kept in Latin script everywhere because users type it that way.
- Have a native speaker do a 5-minute review before publishing — Play penalizes
  nothing for machine-feel, but natural copy converts better.
